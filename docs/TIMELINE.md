## What "done" means at launch

- All 7 page types live and responsive across mobile, tablet, and desktop
- 5 content areas connected to the CMS so the team can edit team profiles, testimonials, milestones, quote-form settings, and key stats without a developer
- Quote form delivers each submission directly to the ops inbox with spam protection active
- Performance: Lighthouse score ≥ 95 on the home page
- Accessibility: WCAG AA color contrast site-wide, full keyboard navigation
- Images optimized for fast loading on every connection

Post-launch: a 6-week warranty period for bug fixes, content updates, and minor design refinements as the team uses the site.
